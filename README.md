# Z_SALES_ALV_REPORT – SAP ABAP ALV Dashboard

Custom SAP ABAP ALV report for Sales Order analysis built using classical ALV framework.

## Features

- Multi-table JOIN (VBAK, VBAP, KNA1)
- Selection-screen filtering
- Traffic-light status indicators
- Customer-wise subtotals
- Variant layout management
- Drill-down navigation to VA03
- Performance-safe record limiter

## Tech Stack

ABAP 7.4  
SAP ECC / S4HANA  
REUSE_ALV_GRID_DISPLAY  
SE38 / SE11 / SE16
